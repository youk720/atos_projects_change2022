左上のタブで、放送やメロディの設定画面を切り替える。それ以降の使い方は以下を...

自動放送部:使い方;
1, index.htmlを開く(Chromeなどで起動)
2, 画面上で行き先や、番線などをそれぞれ選択(愛称設定部分は未実装のため、選択しても音源はセットされない)
※行先設定時に"行先"からの選択だけを流す場合、単独(連続)で接近時に"xx行きが、まいります", 単独(非連続)は"xx、行きが、まいります"と流れる
2つ(行きわけ)は"行先""行先2"行きが...などと流れ、2つ(山手用)は"行先・行先2方面ゆきが...(パーツにより"行先2、方面行きが"と別れる場合あり)"と流れる
なお、上記のパターンは予告でも対応
3, 接近ボタン、またはRキー(使用マシンにより変わる可能性あり)で接近放送用のパーツがセットされる。
予告ボタン、またはWキーで予告用のパーツがセットされる。なおボタンの上の、放送種別(予告用)を切り替えることで、放送分を若干変化できる
また再生順入力へ、全パーツ用意にあるパーツであれば、手動で入力が可能(なお前述・後述のキーボード操作により、"強制的に"作動する場合があり)
4, 再生またはGキーでセットされた音源が再生される

発車メロディ部:使い方
1, PC内で予め用意した、メロディ音源のパスを"メロディ音源手動入力"へ入力し、決定を押す
2, ON(Eキー), OFF(Dキー)でメロディの再生・停止操作をする。また、それとは別に、Startを押してメロディが流れるが、Startのクリックを押しっぱなしから、離して約10秒後に、放送が流れる仕様がついている。
※:メロディの切れ方は、通常で途中切りができ、立川式で、OFF操作時から発車放送が流れ、
ノン被り式でメロディのループがなくなり、コーラス鳴り終わると発車放送が流れる仕様。
発車放送設定:
"x番線", "x番線の"で、"x番線(の)　ドアが閉まります。ご注意ください"
"路線名付き"で"x番線の、(自動放送タブで、選択された路線名)、ドアが閉まります。ご注意ください"
"仙台式"で、"x番線から、(路線名)(種別)(行先A)(行先B)行きが 発車いたします。ドアが閉まります。ご注意ください"
のように流れる

おまけとして、ノイズのチェックが入ればメロディにノイズが被リや、禁煙放送(xキー)で"当駅では、終日禁煙..."が流れる
